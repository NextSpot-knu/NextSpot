### 실패 시 폴백 (관광객 시나리오)

- **카카오맵 길안내 미전환(PC/미설치)**: 상세 카드의 좌표·거리 텍스트로 대체 설명, 필요 시 캡처 화면 준비.
- **백엔드/네트워크 장애**: 추천 API 실패 시 프런트 로컬 미러 계산으로 화면이 무너지지 않고 유지되는
  점을 그대로 시연 멘트에 포함("백엔드가 잠깐 흔들려도 사용자 화면은 끊기지 않습니다"). 완전 단절 시
  사전 녹화 백업 영상으로 즉시 전환.
- **TourAPI 상세 데이터 없음**: 캐시된 최근 적재 데이터로 계속 진행(실시간 호출이 아니라 사전 배치임을
  자연스럽게 멘트에 녹인다).

  ①↔②로 자연스럽게 바꿔 진행한다.

## 3. 발표 운영 메모

- 관광객 시나리오는 **음성 성공 경로**를 기본으로 하되, 리허설에서 인식률이 불안정하면 처음부터
  텍스트 검색 경로로 전환해도 좋다(폴백이 아니라 정식 경로로 소개).
- 관리자 시나리오의 "예시 추이(데모)" 발화는 반드시 포함한다 — 생략 시 실측으로 오인되어 정직성
