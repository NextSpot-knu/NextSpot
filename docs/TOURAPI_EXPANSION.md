> 원칙: **혼잡 분산이라는 고유 자산을 강화하는 데이터 우선** → 심사 'TourAPI 활용 적절성' 어필 → 상용 확장.

## Tier 3 — 상용 단계 (정직한 한계 명시)
- **빅데이터 방문자수**(15101972, KT+SKT): **시군구 단위**라 POI 혼잡 대체 불가 → 용도는 ① 우리 예측 모델의 도시 단위 추이 교차검증 ② B2G 성과 리포트에 외부 데이터 문단(저난도 가치).
- **POI 단위 실시간 혼잡도는 TourAPI 에 없음** — KT TrIP·SK 지오비전 등 유료 상용 API 영역(지자체 계약 사안). 해수욕장 혼잡도 신호등(15061784)이 공공 선례.
- `searchStay2` 숙박: 코스+숙박 제휴(커미션)는 상용화 이후.
